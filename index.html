Guide des Classes Importantes en Bootstrap 5

Découvrez les classes principales et leur usage en responsive

1. Système de Grille

Bootstrap utilise un système de grille basé sur 12 colonnes. Vous pouvez contrôler la taille des colonnes avec des classes comme .col-md-* où md est un point de rupture (responsive) et * est la largeur de la colonne.

Responsivité : Utilisez .col-sm-*, .col-md-*, .col-lg-* pour ajuster la taille selon la largeur de l’écran.

Colonne 1
Colonne 2
Colonne 3
<div class="row">
    <div class="col-sm-6 col-md-4 bg-warning">Colonne 1</div>
    <div class="col-sm-6 col-md-4 bg-info">Colonne 2</div>
    <div class="col-md-4 bg-success">Colonne 3</div>
</div>

2. Boutons

Bootstrap propose différents styles de boutons prédéfinis :

  • .btn → Classe de base
  • .btn-primary, .btn-danger, .btn-success → Styles de couleurs
  • .btn-lg, .btn-sm → Tailles
  • .btn-outline-* → Boutons avec contours
<button class="btn btn-primary">Principal</button>
<button class="btn btn-danger btn-lg">Grand Bouton</button>
<button class="btn btn-success btn-sm">Petit Bouton</button>
<button class="btn btn-outline-warning">Bouton Contour</button>

3. Alertes

Les alertes permettent d'afficher des messages colorés pour informer l'utilisateur. Utilisez .alert avec des couleurs comme .alert-success, .alert-danger.

Succès ! Action réalisée.
Erreur ! Problème détecté.
<div class="alert alert-success">Succès !</div>
<div class="alert alert-danger">Erreur !</div>

4. Cartes

Les cartes permettent de structurer des blocs d’information avec un design soigné.

Responsivité : Ajoutez .card-img-top pour une image responsive en haut.

Titre de la carte

Exemple de contenu.

<div class="card shadow">
    <img src="image.jpg" class="card-img-top">
    <div class="card-body">
        <h5 class="card-title">Titre</h5>
        <p class="card-text">Contenu</p>
    </div>
</div>

Guide des Barres de Navigation Bootstrap

Découvrez les classes essentielles pour créer des navigations modernes et responsives

1. Navbar de Base

La classe .navbar permet de créer une barre de navigation. Utilisez .navbar-expand-lg pour rendre le menu responsive.

MonSite
  • Accueil
  • Services
  • Contact
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">MonSite</a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navbarBasic">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarBasic">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#">Accueil</a></li>
                <li class="nav-item"><a class="nav-link" href="#">Services</a></li>
                <li class="nav-item"><a class="nav-link" href="#">Contact</a></li>
            </ul>
        </div>
    </div>
</nav>

2. Navbar avec Logo et Bouton

Ajoutez un logo et un bouton avec .btn dans la barre de navigation.

MonLogo
  • Accueil
  • Blog
S'inscrire 
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">
            <img src="logo.png" class="me-2"> MonLogo
        </a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navbarLogo">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarLogo">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#">Accueil</a></li>
                <li class="nav-item"><a class="nav-link" href="#">Blog</a></li>
            </ul>
            <a href="#" class="btn btn-primary ms-3">S'inscrire</a>
        </div>
    </div>
</nav>

3. Navbar Fixe en Haut

Utilisez .fixed-top pour une barre de navigation fixe.

<nav class="navbar navbar-dark bg-primary fixed-top">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Fixe en Haut</a>
    </div>
</nav>

4. Navbar avec Menu Déroulant

Utilisez .dropdown pour ajouter des sous-menus.

Menu Déroulant
  • Catégories
    • Catégorie 1
    • Catégorie 2
<a class="nav-link dropdown-toggle" href="#" data-bs-toggle="dropdown">Catégories</a>
<ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Catégorie 1</a></li>
    <li><a class="dropdown-item" href="#">Catégorie 2</a></li>
</ul>

formulaires

1. Structure de Base

Bootstrap fournit une mise en forme de base pour les formulaires en utilisant la classe .form-control pour les champs de saisie et .form-label pour les libellés. 

    <form>
        <div class="mb-3">
            <label class="form-label">Nom</label>
            <input type="text" class="form-control" placeholder="Entrez votre nom">
        </div>
        <button class="btn btn-primary">Envoyer</button>
    </form>

2. Types de Champs

Bootstrap prend en charge plusieurs types de champs avec .form-control, qui adapte automatiquement leur apparence. 

    <input type="email" class="form-control" placeholder="exemple@mail.com">
    <input type="password" class="form-control" placeholder="********">
    <input type="date" class="form-control">
    <input type="file" class="form-control">

3. Champs avec Icônes

Utilisez .input-group pour ajouter des icônes ou du texte avant/après un champ.

@ 
    <div class="input-group">
        <span class="input-group-text">@</span>
        <input type="text" class="form-control" placeholder="Votre pseudo">
    </div>

4. Boutons Radio & Cases à Cocher

Utilisez .form-check pour aligner correctement les cases et boutons radio.

    <div class="form-check">
        <input type="checkbox" class="form-check-input" id="check1">
        <label for="check1" class="form-check-label">J'accepte les conditions</label>
    </div>
    <div class="form-check">
        <input type="radio" class="form-check-input" name="gender" id="radio1">
        <label for="radio1" class="form-check-label">Homme</label>
    </div>

5. Formulaires Responsives

Utilisez .row et .col-md-* pour organiser les champs en colonnes adaptatives.

    <div class="row">
        <div class="col-md-6">
            <input type="text" class="form-control" placeholder="Votre prénom">
        </div>
        <div class="col-md-6">
            <input type="text" class="form-control" placeholder="Votre nom">
        </div>
    </div>

Guide du Responsive Design avec Bootstrap 5

Découvrez comment créer des mises en page adaptatives

1. Le Système de Grille

Bootstrap utilise un système de grille basé sur 12 colonnes. Les classes .col-* permettent d’adapter la mise en page selon la taille de l’écran.

Exemple : Colonnes qui changent selon la taille de l’écran.

Colonne 1
Colonne 2
Colonne 3
Colonne 4
<div class="row">
    <div class="col-sm-6 col-md-4 col-lg-3 bg-warning">Colonne 1</div>
    <div class="col-sm-6 col-md-4 col-lg-3 bg-info">Colonne 2</div>
    <div class="col-sm-12 col-md-4 col-lg-3 bg-success">Colonne 3</div>
    <div class="col-sm-12 col-lg-3 bg-danger">Colonne 4</div>
</div>

2. Images Responsives

Utilisez .img-fluid pour que les images s’adaptent automatiquement à leur conteneur.

<img src="image.jpg" class="img-fluid" alt="Image Responsive">

3. Alignements et Visibilité

Bootstrap permet d’afficher ou cacher des éléments selon la taille de l’écran.

Visible seulement en petit écran
Visible seulement en écran moyen
Visible seulement en grand écran 
<div class="d-none d-sm-block d-md-none">Visible en petit écran</div>
<div class="d-none d-md-block d-lg-none">Visible en écran moyen</div>
<div class="d-none d-lg-block">Visible en grand écran</div>

4. Navigation Responsive

Utilisez .navbar-expand-lg pour une barre de navigation adaptative.

Navbar Responsive
  • Accueil
  • Blog
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Navbar Responsive</a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navbarResponsive">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarResponsive">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#">Accueil</a></li>
                <li class="nav-item"><a class="nav-link" href="#">Blog</a></li>
            </ul>
        </div>
    </div>
</nav>

5. Tableaux Responsives

Utilisez .table-responsive pour rendre un tableau adaptable sur mobile.

# Nom Email
1 Jean Dupont jean@mail.com
2 Marie Curie marie@mail.com 
<div class="table-responsive">
    <table class="table table-bordered">
        <thead class="table-dark">
            <tr>
                <th>#</th>
                <th>Nom</th>
                <th>Email</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>1</td>
                <td>Jean Dupont</td>
                <td>jean@mail.com</td>
            </tr>
        </tbody>
    </table>
</div>
